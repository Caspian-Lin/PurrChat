不要使用内置的 `WebSearch` 工具，改用外接的 exa MCP 工具：
- 搜索：使用 `mcp__exa__web_search_exa`
- 网页抓取：使用 `mcp__exa__web_fetch_exa`

**Why:** 用户偏好使用 exa MCP 提供的搜索能力，结果质量更好。

**How to apply:** 每次需要进行网络搜索或抓取网页内容时，优先选择 exa MCP 工具而非内置 WebSearch。
